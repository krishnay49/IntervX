# Contributing to IntervX

Thank you for considering contributing to IntervX! We appreciate your interest and support.

## How to Contribute

1. **Fork the repository**: Click the "Fork" button at the top right corner of the repository page.

2. **Clone your fork**:
```bash
   git clone https://github.com/krishnay49/IntervX.git
```

3. **Navigate to the project directory**:
```bash
   cd IntervX
```

4. **Create a new branch**:
```bash
   git checkout -b your-feature-branch
```

5. **Make your changes**: Add features or fix bugs.

6. **Commit your changes**:
```bash
   git commit -m "Describe your changes"
```

7. **Push your branch**:
```bash
   git push origin your-feature-branch
```

8. **Create a pull request**: Go to the repository and open a pull request.

## Code Style
- Follow existing code structure and naming conventions
- Use meaningful commit messages
- Test your changes before submitting
