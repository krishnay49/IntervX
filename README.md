## 🔍 Overview
IntervX allows users to practice technical and behavioral interview questions in an interactive AI-powered environment. The app generates custom questions based on job role, tech stack, and experience level, and delivers instant feedback on answers.

## ✨ Features
- **AI-Driven Questions** — Custom interview questions generated by Gemini AI based on your profile
- **User Authentication** — Secure login and signup using Clerk
- **Real-Time Feedback** — Instant AI-powered analysis and rating of your answers
- **Webcam Support** — Practice with live webcam during mock interviews
- **Interview History** — Track all your past interviews and progress
- **Dark Theme UI** — Clean, modern dark interface built with Tailwind CSS

## 🛠️ Technologies Used
- **[Next.js](https://nextjs.org/)** — Framework for server-rendered React applications
- **[React](https://reactjs.org/)** — JavaScript library for building user interfaces
- **[Tailwind CSS](https://tailwindcss.com/)** — Utility-first CSS framework
- **[Drizzle ORM](https://drizzle.team/)** — ORM for managing database interactions
- **[Neon](https://neon.tech/)** — Serverless PostgreSQL database
- **[Gemini AI](https://ai.google.dev/)** — AI API for generating questions and analyzing answers
- **[Clerk](https://clerk.dev/)** — Authentication and user management

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm
- PostgreSQL database (e.g. Neon)

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/krishnay49/IntervX.git
   cd IntervX
```

2. Install dependencies:
```bash
   npm install
```

3. Create a `.env.local` file in the root directory:
```env
   NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
   CLERK_SECRET_KEY=your_clerk_secret_key

   NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
   NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up

   NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key

   NEXT_PUBLIC_DRIZZLE_DB_URL=your_postgres_connection_string
```

4. Set up the database by running this SQL in your Neon SQL Editor:
```sql
   CREATE TABLE "MockInterview" (
       id SERIAL PRIMARY KEY,
       "jsonMockResp" TEXT NOT NULL,
       "jobPosition" VARCHAR NOT NULL,
       "jobDesc" VARCHAR NOT NULL,
       "jobExperience" VARCHAR NOT NULL,
       "createdBy" VARCHAR NOT NULL,
       "createdAt" VARCHAR,
       "mockId" VARCHAR NOT NULL
   );

   CREATE TABLE "userAnswer" (
       id SERIAL PRIMARY KEY,
       "mockId" VARCHAR NOT NULL,
       "question" VARCHAR NOT NULL,
       "correctAns" TEXT,
       "userAns" TEXT,
       "feedback" TEXT,
       "rating" VARCHAR,
       "userEmail" VARCHAR,
       "createdAt" VARCHAR
   );
```

5. Start the development server:
```bash
   npm run dev
```

6. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🔑 Environment Variables

| Variable | Description |
|---|---|
| `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` | Clerk publishable key |
| `CLERK_SECRET_KEY` | Clerk secret key |
| `NEXT_PUBLIC_GEMINI_API_KEY` | Google Gemini AI API key |
| `NEXT_PUBLIC_DRIZZLE_DB_URL` | PostgreSQL connection string |
